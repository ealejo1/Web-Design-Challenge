Summary: Latitude vs. X

The purpose of this project is to visualize how weather changes as a person approaches closer to the equator. In order to prove the analysis, I extracted a dataset from Open Weather API consisting of 500 cities . Upon assembling the dataset, I utilized Matplotlib to plot a variety of weather aspects vs. latitude. The following weather factors were explored: humidity, cloudiness, maximum temperature and wind speed. As part of the analysis this site has the source data and visualizations, as well as depictions and explanations of any seen and proven correlations.
